ローカルLLMでExcelを自然言語編集
ここにファイルをドラッグ＆ドロップ、またはクリックして選択
このツールでできること / できないこと
✅ できること
  • 値の編集・一括置換・条件付きの書き換え
  • 行・列の追加 / 削除 / 並べ替え
  • 集計（合計・平均・最大最小・件数）と数式の挿入
  • 書式：太字・斜体・文字色・背景色・罫線・配置
  • 表示形式（カンマ・通貨・日付など）、列幅・行高
  • セルの結合 / 解除、セルにコメント追加
  • 複数シートの操作（シート追加・シート間コピー）
  • 条件付き書式、Excelテーブル化、基本的なグラフ（棒・折れ線・円）
🚫 できないこと
  • ピボットテーブルの作成
  • マクロ / VBA の作成・実行（既存マクロは保持のみ）
  • 画像・図形の挿入
  • 数式の「計算結果の値」表示（数式は入りますが、計算はExcelで開いた時。プレビューは数式の文字列）
  • 外部データ取得・Web連携・他システム接続
  • 複数ファイルの結合（1回につき1ファイル）
  • 20MBを超えるファイル
  • 元ファイルにピボット・画像など特殊機能がある場合、保存時に失われることがあります
  • 複雑すぎる多段の指示は失敗することがあります（指示は分割推奨）
